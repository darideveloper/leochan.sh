# Proposal: Update Hero Section Title to Portfolio Owner's Name

The current hero section title displays "INFRASTRUCTURE & Security." to reflect the owner's professional field. To shift towards stronger personal branding, this change replaces that text with the owner's name, **LEONARD-ANTON**, presented in a single, visually dominant line.

## Key Changes
- **Personal Branding:** Replace generic "INFRASTRUCTURE" text with the owner's name "LEONARD-ANTON".
- **Simplified Layout:** Remove the "& Security." tagline below the main title to create a cleaner, more focused headline.
- **Responsive Typography:** Ensure the long name fits on a single line across all viewports (mobile, tablet, desktop) without breaking or overlapping inappropriately.

## Goals
- **Clarity:** Immediately identify the site owner.
- **Impact:** Maintain the massive, bold typography style that defines the portfolio's aesthetic.
- **Responsiveness:** Ensure the name scales gracefully down to mobile screens.
