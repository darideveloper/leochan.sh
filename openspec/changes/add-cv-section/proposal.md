# Proposal: Add Professional CV Section

## Problem
The current portfolio showcases projects and general skills but lacks a comprehensive, structured view of Léonard-Anton's professional background (detailed education, specific technical certifications, aeronautical skills, and detailed experience). Recruiters and collaborators need a "one-stop" section that mirrors a formal CV while maintaining the site's high-tech, cyber-glass aesthetic.

## Proposed Solution
Introduce a new `CV` section in the main page (accessible via navigation) that presents Leonard-Anton's profile in a structured, responsive, and visually cohesive way. This section will reuse existing "Cyber-Glass" styling and typography to ensure it feels like an integrated part of the "Nekocorp" ecosystem.

### Key Features
- **Responsive CV Layout:** A multi-column desktop layout that stacks gracefully on mobile.
- **Categorized Information:** Clear separation between Experience, Education, Technical Skills, and unique traits like Aeronautical Skills.
- **Interactive Elements:** Reuse `TypingText` for section headers and potentially `LetterGlitch` for subtle background accents.
- **Navigation Integration:** Add a `_RESUME` (or `_CV`) link to the main navigation bar.
- **Contact Shortcuts:** Quick links for Email, Tel, and LinkedIn within the CV header.

## Impact
- **Professionalism:** Provides a complete professional profile directly on the site.
- **User Experience:** Allows users to quickly find specific qualifications without leaving the page.
- **SEO:** Adds rich, keyword-heavy content related to Cybersecurity, Infrastructure, and Engineering.
