# Proposal: Add Powered By DariDeveloper Link

## Problem Statement
The website currently does not attribute the developer in the footer. Adding a "Powered By" link provides professional attribution and a way for users to contact the developer.

## Goals
- Add a "Powered By DariDeveloper" link to the footer.
- Ensure the link points to `https://www.darideveloper.com/`.
- The link should open in a new tab.

## Non-Goals
- Changing the overall layout of the footer.
- Adding other links or social icons.

## Proposed Changes
- Modify `src/components/footer.astro` to include the attribution link.
- Update `openspec/specs/ui-elements/spec.md` with the new requirement.

## Benefits
- Clear attribution for the developer.
- Professional credit.
