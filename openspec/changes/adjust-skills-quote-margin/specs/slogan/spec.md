# slogan Specification (Delta)

## MODIFIED Requirements
### Requirement: Display Professional Slogan
The professional slogan highlights the engineer's expertise in infrastructure automation and cybersecurity.

#### MODIFIED Scenario: Slogan in Skills Section
- **Given** the user visits the home page on desktop.
- **When** the **Skills** section is rendered.
- **Then** the slogan container **MUST** have a top margin (`lg:mt-12` or similar) to achieve better visual centering relative to the adjacent `SkillsList`.
